# Google Ads Demo

## Description:
This project is an illustrative example for showcasing Google Ads.

  - Within this sample project, you will find various ad formats, including:
    - App Open Ads
    - Banner Ads
    - Interstitial (Full-Screen) Ads
    - Rewarded Ads
    - Rewarded Interstitial Ads
